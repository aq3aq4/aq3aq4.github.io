# Compute Engine 

## 인스턴스 생성

* **인스턴스 이름**: 하고싶은이름으로 해주세요.
* **인스턴스 리전, 영역** : 서울, 영역 A (asia-northeast3-a)
* **인스턴스 이미지 선택** : Linux
* **머신 유형** : e2-medium(vCPU)
* **부팅 디스크** : Ubuntu 20.04 LTS
* **표준 영구 디스크**: 64GB

### ssh 키를 이용한 접속    
* 접속 키 생성
* 생성한 키의 공개키를 Cloud Console에서 등록
```shell
ssh-keygen -t rsa -C "aq3aq4@google.com"
```
